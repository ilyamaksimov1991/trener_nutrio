# Питание на массу — мини-сайт

Статический сайт (HTML+JS, без сборки). Открой `index.html` — это главная.

## Страницы
- `index.html` — хаб со ссылками
- `konstruktor_dnya.html` — конструктор дня (живой расчёт КБЖУ, переключатель трен/отдых, сохранение комбинаций, список покупок)
- `menu_nedeli.html` — готовое меню на неделю + список продуктов
- `menu_alternativy.html` — альтернативные «тестовые» дни
- `smuzi.html` — 8 вариантов смузи (КБЖУ + граммовка)
- `zavtraki.html` — 8 вариантов завтраков (КБЖУ + граммовка)
- `produkty.html` — правка продуктов (CRUD), экспорт `products.js`
- `menu_week_v2.html` — печатная версия меню (для холодильника)
- `products.js` — единый источник КБЖУ (можно править вручную)

Данные (твои сборки, правки продуктов) хранятся в браузере (localStorage) — у каждого устройства/браузера свои. Личных данных на сайте нет, можно спокойно делиться с женой.

## Посмотреть локально
Просто открой `index.html` двойным кликом. Все файлы должны лежать рядом (в одной папке).

## Залить на GitHub Pages
1. Создай репозиторий на GitHub (например, `nutrio`).
2. Загрузи в него ВСЁ содержимое этой папки `site/` (можно перетащить файлы в веб-интерфейсе GitHub: кнопка **Add file → Upload files**), затем **Commit**.
3. В репозитории: **Settings → Pages**.
4. **Source:** Deploy from a branch. **Branch:** `main`, папка `/ (root)`. **Save**.
5. Через ~1 минуту сайт будет доступен по адресу:
   `https://<твой-логин>.github.io/<имя-репозитория>/`
   (например, `https://ivan.github.io/nutrio/`).

Файл `.nojekyll` уже добавлен — он отключает обработку Jekyll, чтобы всё рендерилось как есть.

## Через терминал (если удобнее git)
```bash
cd site
git init
git add .
git commit -m "nutrition site"
git branch -M main
git remote add origin https://github.com/<логин>/<репо>.git
git push -u origin main
# затем включить Pages в Settings → Pages (branch main, /root)
```

## Обновление меню/продуктов
- Правь продукты прямо на странице `produkty.html` → «Экспорт products.js» → замени им файл `products.js` в репозитории (Upload files) → коммит. Сайт обновится.
- Или редактируй `products.js` как текст и заливай.

КБЖУ — ориентир (партии ВкусВилла меняются, точное на упаковке). Для лога питания используется FatSecret. Не медицинское назначение.
